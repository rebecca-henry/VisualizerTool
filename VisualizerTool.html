Scene Description Visualizer

About this tool ▾
This is a visualization tool to help explain the LLM-generated scene descriptions. Select one of 6 preset scene descriptions. These were generated by an LLM using my spatial framework. Each scene description describes a different scene. The reference photo on the left was shown to human annotators, who wrote 5 captions. These captions were given to the LLM, along with my spatial framework. The LLM generated the scene description. You can explore the data in the middle and right columns. To get an even better idea of the information contained in a scene description, you can plug in an OpenAI API key and generate an image! (your key will not be stored anywhere)
